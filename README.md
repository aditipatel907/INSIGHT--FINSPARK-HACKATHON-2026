# INSIGHT
### Insider Security & Intelligent Governance for High-Privilege Threats

> Securing Trust Through Intelligent Insight.

INSIGHT is a cybersecurity solution designed for the **FinSpark'26 Hackathon** to address insider threats involving privileged users in the banking sector.

## About the Project

INSIGHT combines:

-  Duress (Emergency) Password Authentication
-  Behaviour Analytics
-  Explainable AI-inspired Risk Analysis
-  Business Context Verification
-  Security Operations Center (SOC) Monitoring
-  Audit Logs & Investigation Support

The solution enables employees under coercion to silently alert the Security Operations Center (SOC) while appearing to authenticate normally, helping organizations detect and respond to insider threats in real time.
